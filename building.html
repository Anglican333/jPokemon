Building

If youre trying to compile/build this API yourself from the source, you can do so using whatever method youd like. There are no dependencies at this time, so theres no need to worry about them. The project itself uses Maven, the use of which is outlined below. If you want to generate the site or the Javadocs in the same format they are featured here, things are little more complicated: the comments in the .java files are written in Markdown syntax, and to properly display them you will need the Markdown doclet.

Building with Maven

Make sure you have Maven 2 or 3 installed on your system. You can check this by entering mvn --version on the command line. After you have acquired Maven, navigate to the directory you wish to place the source code and enter the following on the command line:

cd /path/to/repo
git clone https://github.com/atheriel/jPokemon.git
cd jPokemon
mvn clean install

The project is now installed on your local maven repository, and the .jar file can be found in the target directory.
